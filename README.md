# 简介

此仓库包括我所做过的来自各大题库的 OJ 算法题的题解，这些题库包括（依照使用度依次递减）：

* 浙江大学计算机程序设计能力考试（PAT）题库，
* UVa 题库，
* Codeup 题库，
* 洛谷题库，
* 北京大学 OJ 题库（POJ），
* 安徽科技学院 ACM 题库（AOJ）
* ...

目前在主攻 PAT，完成度已过 55.6%，其中部分题解提供多种方法、以及我给出的测试样例。
